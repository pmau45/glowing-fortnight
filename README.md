# Axiom Canine

A professional website for Axiom Canine, a tactical K9 training and services company.

## Overview

This website showcases Axiom Canine's expertise in canine training, featuring a modern, tactical-inspired design with custom animations and interactive elements.

## Technologies Used

- **HTML5**: Semantic markup structure
- **Tailwind CSS**: Utility-first CSS framework (via CDN)
- **Custom CSS**: Advanced animations, noise overlays, and scroll effects
- **JavaScript**: Interactive features including:
  - Form handling with Netlify Forms integration
  - Smooth scroll animations
  - Intersection Observer for fade-in effects
  - Parallax scrolling effects

## Design Features

### Color Scheme
- **Ranger** (#2F3A25): Primary dark green
- **Mud** (#3B3024): Secondary dark brown
- **Bone** (#EBE9E1): Light text/background
- **Coyote** (#D2B48C): Accent tan
- **Signal** (#FF4500): Highlight red-orange

### Custom Fonts
- **Inter**: Primary sans-serif font
- **Bebas Neue**: Tactical headings
- **Space Mono**: Monospace details

### Visual Effects
- Animated noise texture overlay
- Scan line animation
- Parallax scrolling
- Fade-in animations on scroll
- Border reveal effects
- Custom scrollbar styling

## How to Use

### Viewing Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/pmau45/glowing-fortnight.git
   cd glowing-fortnight
   ```

2. Open `index.html` in your web browser:
   ```bash
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   
   # On Windows
   start index.html
   ```

### Deploying

This is a static website that can be hosted on any web server or static hosting service:

- **GitHub Pages**: Enable in repository settings
- **Netlify**: Drag and drop deployment (form submissions supported)
- **Vercel**: Connect repository for automatic deployment
- **Traditional Web Hosting**: Upload `index.html` to your server

## Form Integration

The contact form is configured for Netlify Forms integration with:
- Spam protection via honeypot field
- AJAX submission with loading states
- Success/error feedback messages
- Automatic field reset on successful submission
